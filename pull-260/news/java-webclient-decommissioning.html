Java webclient decommissioning

 Martinp23 on 2010-08-04

Fol­low­ing our suc­cess­ful switch of cloak­ing on our web gate­way (http://we­bchat.freen­ode.net) to show the full IP ad­dress of con­nect­ing users (see this blog post), we have de­cid­ed to tran­si­tion our old and rel­a­tive­ly un­used Java client (pjIRC) to our we­bchat ser­vice. This will be done via a HTTP redi­rect.

Only around 30 users at a time can be found from the java client, hence as time goes on it makes less and less sense to con­tin­ue to sup­port this plat­form. We'll be de­com­mis­sion­ing the Java client on Sun 8th Au­gust.

Oth­er pjIRC in­stances which con­nect to freen­ode will be un­af­fect­ed. We are sim­ply re­mov­ing our ver­sion of the pro­gram.

If you've any con­cerns, queries or com­ments we'd love to hear from you ei­ther in #freen­ode or via sup­port at freen­ode.net.
